Implementation of the Hybrid Fictitious Domain-\/\+Immersed Boundary (H\+F\+D\+IB) coupled with the Discrete Element Method (D\+EM) by

Martin Isoz (\href{https://github.com/MartinIsoz}{\texttt{ https\+://github.\+com/\+Martin\+Isoz}}) Martin Kotouč Šourek (\href{https://github.com/MartinKotoucSourek}{\texttt{ https\+://github.\+com/\+Martin\+Kotouc\+Sourek}}) Ondřej Studeník (\href{https://github.com/OStudenik}{\texttt{ https\+://github.\+com/\+O\+Studenik}}) and Petr Kočí (\href{http://vscht.cz//monolith/}{\texttt{ http\+://vscht.\+cz//monolith/}})

\href{https://github.com/fmuni/openHFDIB}{\texttt{ https\+://github.\+com/fmuni/open\+H\+F\+D\+IB}}

by Federico Municchi

The D\+EM implementation for arbitrarily shaped solids into Open\+F\+O\+AM is original.\hypertarget{md_README_autotoc_md1}{}\doxysection{Code capabilities}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item simulation settings is defined in H\+F\+D\+I\+B\+D\+E\+Mdict (see D\+O\+C\+U\+M\+E\+N\+T\+A\+T\+I\+ON)
\item simulations with either spherical or S\+T\+L-\/defined particles (see H\+F\+D\+I\+B\+D\+E\+M/geom\+Models)
\item simulations of two-\/phase (solid-\/fluid) flow (see pimple\+H\+F\+D\+I\+B\+Foam)
\item simulations of solid phase in standard D\+EM mode (see H\+F\+D\+I\+B\+D\+E\+M\+Foam)
\item adaptive mesh refinement based on the particles position
\item spring-\/dashpot contact model based on particle elastic modulus and a damping constant (see H\+F\+D\+I\+B\+D\+E\+M/contact\+Models)
\item extension to contact model for inclusion of adhesive forces
\item multiple solid phase initialization options such as random spatial distribution of uniformly sized bodies (see H\+F\+D\+I\+B\+D\+E\+M/add\+Models)
\item instructive tutorials for a single particle falling through a fluid and for interaction between a particle and a complex-\/shaped impeller (see Tutorials)
